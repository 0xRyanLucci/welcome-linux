# Welcome Linux

A vaporwave-themed interactive directory navigator for your Linux terminal.

## Install

```bash
bash <(curl -sL https://raw.githubusercontent.com/0xRyanLucci/welcome-linux/master/install.sh)
```

The installer will prompt you to configure:
- Directory to scan (~/code, ~/projects, ~/dev, or custom)
- Number of directories to show
- Auto-run on shell start

Run again anytime to update or reconfigure.

## Usage

Runs automatically on shell start. Or invoke manually:

```bash
welcome
```

## Controls

| Key | Action |
|-----|--------|
| Up/Down | Navigate |
| Enter | CD to directory |
| C | Open Claude Code |
| Esc | Exit |

## Features

- Shows most recently modified directories from your code folder
- Vaporwave ASCII art aesthetic
- Interactive arrow key navigation
